---
title: Heading
component: Heading
description: Useful for rendering headlines.
---

```js live
<Heading>Heading component</Heading>
```

### children

<Prop>ReactNode</Prop>

The content to render inside the Heading.

### size

<Prop isOptional default="2">
  0 | 1 | 2 | 3 | 4 | 5
</Prop>

There are 6 sizes to choose from. Each size handles its own `font-size`, `letter-spacing` and `text-indent` accordingly.

```js live
<Heading size={0}>Heading component</Heading>
<Heading size={1} mt={1}>Heading component</Heading>
<Heading size={2} mt={1}>Heading component</Heading>
<Heading size={3} mt={1}>Heading component</Heading>
<Heading size={4} mt={1}>Heading component</Heading>
<Heading size={5} mt={1}>Heading component</Heading>
```

### weight

<Prop isOptional default="400">
  400 | 500
</Prop>

There are 2 weight options to choose from.

```js live
<Heading fontWeight="400">Heading component</Heading>
<Heading fontWeight="500">Heading component</Heading>
```

### truncate

<Prop isOptiona default="false">
  Boolean
</Prop>

When `true` this will truncate your text based on its maximum width.

```js live
<Box width={100}>
  <Heading truncate>Heading component</Heading>
</Box>
```
