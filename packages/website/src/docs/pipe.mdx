---

```js live
<Pipe />
```

### variant

<Prop isOptional default="normal">
  normal | large | fluid
</Prop>

There are 3 variants to choose from

```js live
<Pipe variant="normal" />
<Pipe variant="large" mt={6}/>
<Box height={50}>

<Pipe variant="fluid" mt={6}/>
</Box>
```
