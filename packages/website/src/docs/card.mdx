---
title: Card
component: Card
description: Useful to group information and actions about a single subject.
---

```js live
<Card maxWidth={275}>
  <Heading size={1} fontWeight={500}>
    Card heading
  </Heading>
  <Text as="p" mt={2} mb={4} size={3}>
    Card copy
  </Text>
  <Button>Action</Button>
</Card>
```

### children

<Prop>ReactNode</Prop>

The content to render inside the Card.

### variant

<Prop isOptional default="normal">
  normal | shadow | ghost
</Prop>

There are 3 variants to choose from.

```js live
<Card maxWidth={275} variant="shadow">
  <Heading size={1} fontWeight={500}>
    Shadow card
  </Heading>
  <Text as="p" mt={2} mb={4} size={3}>
    Card copy
  </Text>
  <Button>Action</Button>
</Card>
```

```js live
<Card maxWidth={275} variant="ghost">
  <Heading size={1} fontWeight={500}>
    Ghost card
  </Heading>
  <Text as="p" mt={2} mb={4} size={3}>
    Card copy
  </Text>
  <Button>Action</Button>
</Card>
```
