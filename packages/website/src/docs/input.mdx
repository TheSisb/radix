---
title: Input
component: Input
description: Useful for allowing users to enter, update and select text.
---

```js live
<Input placeholder="Your e-mail address" />
```

### size

<Prop isOptional default="0">
  0 | 1
</Prop>

There are 2 sizes to choose from.

```js live
<Input placeholder="Your e-mail address" />
<Input placeholder="Your e-mail address" mt={6} size={1} />
```

### variant

<Prop isOptional default="normal">
  normal | fade | ghost
</Prop>

There are 3 variants to choose from.

```js live
<Input value="your@email.io" variant="normal" />
<Input value="your@email.io" mt={6} variant="fade" />
<Input value="your@email.io" mt={6} variant="ghost" />
```
