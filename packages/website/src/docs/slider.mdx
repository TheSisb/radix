---
title: Slider
component: Slider
description: Useful for allowing users to select a value from a range.
---

```js live
<Slider value={50} />
```

### variant

<Prop isOptional default="normal">
  normal | fade
</Prop>

There are 2 variants to choose from.

```js live
<Slider value={50} />
<Slider value={50} variant="fade" mt={6}/>
```
