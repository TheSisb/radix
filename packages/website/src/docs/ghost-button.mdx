---
title: Ghost Button
component: GhostButton
description: Useful to communicate actions to your users via iconography.
---

```js live
<GhostButton>
  <PlusIcon size="15" />
</GhostButton>
```

### children

<Prop>ReactNode</Prop>

The content to render inside the GhostButton.

### size

<Prop isOptional default="0">
  0 | 1
</Prop>

There are 2 size options to choose from.

```js live
<GhostButton>
  <PlusIcon size="15" />
</GhostButton>

<GhostButton size={1} ml={6}>
  <HamburgerIcon size="25" />
</GhostButton>
```

### isActive

<Prop isOptional default="false">
  Boolean
</Prop>

For when your button is active.

```js live
<GhostButton isActive>
  <PlusIcon size="15" />
</GhostButton>
```

### disabled

<Prop isOptional default="false">
  Boolean
</Prop>

For when your button is disabled.

```js live
<GhostButton disabled>
  <PlusIcon size="15" />
</GhostButton>
```
