---
title: Pill Nav
component: PillNav
description: Useful for displaying a list of navigation choices.
---

```js live
<PillNav>
  <Pill>Settings</Pill>
  <Pill>Assets</Pill>
  <Pill>Components</Pill>
  <Pill>Screen</Pill>
</PillNav>
```

### children

<Prop>ReactNode</Prop>

The content to render inside the PillNav.

### pillWidth

<Prop isOpional>CSS.WidthProperty</Prop>

A width value to be passed down to all `Pill` children.

### System props

<SystemProps props={['margin']} />

### Custom props

| Name        | Type            |   Default   | Description                           |
| :---------- | :-------------- | :---------: | :------------------------------------ |
| `active`    | Boolean         |   `false`   | Changes the syling to active state    |
| `disabled`  | Boolean         |   `false`   | Disables the element                  |
| `pillWidth` | ResponsiveValue | `undefined` | Sets a fix width to the Pill elements |
