---
title: Pill Button
component: PillButton
description: Useful to communicate actions to your users within a PillNav.
---

```js live
<PillButton>PillButton</PillButton>
```

### children

<Prop>ReactNode</Prop>

The content to render inside the PillButton.

### pillWidth

<Prop isOpional>CSS.WidthProperty</Prop>

A value to be set as the width property.

### active

For when your PillButton is active.

```js live
<PillButton active>PillButton</PillButton>
```

### disabled

For when your PillButton is disabled.

```js live
<PillButton disabled>Pill</PillButton>
```
