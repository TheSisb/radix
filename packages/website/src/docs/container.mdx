---
title: Container
component: Container
description: Useful for horizontally containing your content.
---

```js live
<Container>
  <Box backgroundColor="blue" height={35} />
</Container>
```

### children

<Prop>ReactNode</Prop>

The content to render inside the Container.

### size

<Prop isOptional default="0">
  0 | 1 | 2
</Prop>

There are 3 sizes to choose from.

```js live
<Container size={1}>
  <Box backgroundColor="blue" height={35} />
</Container>
```

### System props

<SystemProps
  props={['alignSelf', 'flex', 'justifySelf', 'maxWidth', 'margin', 'padding']}
/>
