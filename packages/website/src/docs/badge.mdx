---
title: Badge
component: Badge
description: Useful to emphasise information to the user, works best with single word values.
---

```js live
<Badge>Default</Badge>
```

### children

<Prop>ReactNode</Prop>

The content to render inside the Badge.

### variant

<Prop isOptional default="gray">
  gray | blue | green | red | yellow
</Prop>

There are 5 color variants to choose from.

```js live
<Badge variant="gray">Badge</Badge>
<Badge variant="blue" ml={3}>Badge</Badge>
<Badge variant="green" ml={3}>
  Badge
</Badge>
<Badge variant="red" ml={3}>
  Badge
</Badge>
<Badge variant="yellow" ml={3}>
  Badge
</Badge>
```

### size

<Prop isOptional default="0">
  0 | 1
</Prop>

There are 2 size options to choose from.

```js live
<Badge size={0}>Default</Badge>
<Badge size={1} ml={6}>Large</Badge>
```
