---
title: Menu
component: Menu
description: Useful for displaying a list of choices.
---

```js live
<Menu>
  <MenuItem>
    <Text size={2}>Box</Text>
  </MenuItem>
  <MenuItem variant="selected">
    <Text size={2}>Layout</Text>
  </MenuItem>
  <MenuItem>
    <Text size={2}>Button</Text>
  </MenuItem>
</Menu>
```

### children

<Prop>ReactNode</Prop>

The content to render inside the Menu, ideally these should be [MenuItem](/docs/menu-item/)'s.

### variant

<Prop isOptional default="normal">
  'normal' | 'shadow'
</Prop>

There are 2 variants to choose from.

```js live
<Menu variant="shadow">
  <MenuItem>
    <Text size={2}>Box</Text>
  </MenuItem>
  <MenuItem variant="selected">
    <Text size={2}>Layout</Text>
  </MenuItem>
  <MenuItem>
    <Text size={2}>Button</Text>
  </MenuItem>
</Menu>
```
