```js live
<ToggleButtonGroup value="center" onChange={value => setValue(value)}>
  <ToggleButton value="start">Start</ToggleButton>
  <ToggleButton value="center">Center</ToggleButton>
  <ToggleButton value="end">End</ToggleButton>
</ToggleButtonGroup>
```

```js live
<ToggleButtonGroup value="center" onChange={value => setValue(value)}>
  <ToggleButton value="start" variant="fade">
    Start
  </ToggleButton>
  <ToggleButton value="center" variant="fade">
    Center
  </ToggleButton>
  <ToggleButton value="end" variant="fade">
    End
  </ToggleButton>
</ToggleButtonGroup>
```

## ToggleButtonGroup

### Custom props

| Name       | Type              | Default  | Description                                        |
| :--------- | :---------------- | :------: | :------------------------------------------------- |
| `variant`  | 'normal' \|'fade' | `normal` | Variant to display                                 |
| `value`    | String \| Array   |  `null`  | Value of ToggleButton                              |
| `onChange` | Function          |  `null`  | Callback when a ToggleButton selection has changed |
