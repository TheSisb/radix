---
title: Divider
component: Divider
description: Useful for displaying a visual separator between parts of content.
---

```js live
<Divider />
```

### size

<Prop isOptional>0 | 1 | 2</Prop>

There are 3 size options to choose from. If no `size` is provided, it'll render as full-width.

```js live
<Divider size={0} />
<Divider size={1} mt={6}/>
<Divider size={2} mt={6}/>
```
