---
title: Card Link
component: CardLink
description: Useful to group information about a single subject into a clickable element.
---

```js live
<CardLink maxWidth={275}>
  <Heading size={1} fontWeight={500}>
    Card heading
  </Heading>
  <Text as="p" mt={2} mb={4} size={3}>
    Card copy
  </Text>
  <Text size={2} textColor="grays.5">
    Today
  </Text>
</CardLink>
```

### children

<Prop>ReactNode</Prop>

The content to render inside the CardLink.

### variant

<Prop isOptional default="normal">
  normal | shadow | ghost
</Prop>

There are 3 variants to choose from.

```js live
<CardLink maxWidth={275} variant="shadow">
  <Heading size={1} fontWeight={500}>
    Card heading
  </Heading>
  <Text as="p" mt={2} mb={4} size={3}>
    Card copy
  </Text>
  <Text size={2} textColor="grays.5">
    Today
  </Text>
</CardLink>
```

```js live
<CardLink maxWidth={275} variant="ghost">
  <Heading size={1} fontWeight={500}>
    Card heading
  </Heading>
  <Text as="p" mt={2} mb={4} size={3}>
    Card copy
  </Text>
  <Text size={2} textColor="grays.5">
    Today
  </Text>
</CardLink>
```
