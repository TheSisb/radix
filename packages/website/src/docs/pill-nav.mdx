---

```js live
<PillNav>
  <PillButton>Settings</PillButton>
  <PillButton>Assets</PillButton>
  <PillButton>Components</PillButton>
  <PillButton>Screen</PillButton>
</PillNav>
```

### children

<Prop>ReactNode</Prop>

The content to render inside the PillNav.

### pillWidth

<Prop isOpional>CSS.WidthProperty</Prop>

A width value to be passed down to all `Pill` children.
