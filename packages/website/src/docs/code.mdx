---
title: Code
component: Code
description: Useful for sharing code snippets in a monospaced font.
---

```js live
<Code>console.log('Radix')</Code>
```

### children

<Prop>ReactNode</Prop>

The content to render inside the Code.

### variant

<Prop isOptional default="normal">
  normal | fade
</Prop>

There are 2 variants to choose from.

```js live
<Code variant="normal">normal</Code>
<Code variant="fade" ml={6}>fade</Code>
```
