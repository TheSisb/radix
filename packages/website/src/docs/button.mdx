---
title: Button
component: Button
description: Useful to communicate actions to your users.
---

```js live
<Button>New project</Button>
```

### children

<Prop>ReactNode</Prop>

The content to render inside the Button.

### variant

<Prop isOptional default="gray">
  gray | blue | green | red
</Prop>

There are 6 variants to choose from.

```js live
<Button variant="gray">
  Cancel
</Button>
<Button variant="blue" ml={6}>
  Save
</Button>
<Button variant="green" ml={6}>
  Copy
</Button>
<Button variant="red" ml={6}>
  Delete
</Button>
```

### size

<Prop isOptional default="0">
  0 | 1
</Prop>

There are 2 size options to choose from.

```js live
<Button size={0}>
  Default
</Button>
<Button size={1} ml={6}>
  Large
</Button>
```

### isActive

<Prop isOptional>Boolean</Prop>

For when your button is active.

```js live
<Button isActive>Add</Button>
```

### isWaiting

<Prop isOptional>Boolean</Prop>

When the user is waiting for the server to respond.

```js live
<Button isWaiting>Waiting</Button>
```

### disabled

<Prop isOptional>Boolean</Prop>

For when your button is disabled.

```js live
<Button disabled>New project</Button>
```
