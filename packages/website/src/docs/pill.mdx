---
title: Pill
component: Pill
description: Useful to communicate actions to your users within a PillNav.
---

```js live
<Pill>Pill</Pill>
```

### children

<Prop>ReactNode</Prop>

The content to render inside the Pill.

### pillWidth

<Prop isOpional>CSS.WidthProperty</Prop>

A value to be set as the width property.

### active

For when your Pill is active.

```js live
<Pill active>Pill</Pill>
```

### disabled

For when your Pill is disabled.

```js live
<Pill disabled>Pill</Pill>
```
