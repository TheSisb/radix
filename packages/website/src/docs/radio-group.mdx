---
title: Radio Group
component: RadioGroup
description: Useful for simplifying the usage of Radio elements.
---

```js live
<RadioGroup name="flexDirection">
  <Radio value="row" defaultChecked>
    Row
  </Radio>
  <Radio value="column">Column</Radio>
</RadioGroup>
```

### children

<Prop>ReactNode</Prop>

The content to render inside the RadioGroup, ideally these should be [Radio](/docs/radio/)'s.
